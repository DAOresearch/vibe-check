## Aside Component

<Aside type="tip" title="Pro Tip">
  Use matrix testing to benchmark multiple model configurations in parallel!
</Aside>

<Aside type="caution">
  Agent runs can consume API credits. Always set cost budgets with `toStayUnderCost()`.
</Aside>

<Aside type="note">
  All agent runs are automatically captured with full context - git state, file changes, and tool calls.
</Aside>

## Code Component

<Code code={`vibeTest('benchmark', async ({ runAgent, expect }) => {
  const result = await runAgent({
    agent: defineAgent({ model: 'claude-sonnet-4' }),

  });

  expect(result).toStayUnderCost(2.00);
});`} lang="typescript" title="benchmark.test.ts" />
